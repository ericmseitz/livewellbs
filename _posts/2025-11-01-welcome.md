Welcome! This is the first post for **Live Well Belmont Shore**—a resident-first neighborhood association in Belmont Shore, Long Beach.


This site uses **Jekyll** on **GitHub Pages** with **Bootstrap** for a clean, mobile-friendly design. To publish a new update, just add a Markdown file to the `_posts` folder like this one.


- Meetings and events will be announced here.
- We’ll share safety improvements, surveys, and calls to action.
- Feedback is always welcome.
